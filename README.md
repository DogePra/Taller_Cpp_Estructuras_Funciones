Taller C++ – Estructuras y Funciones

Repositorio que contiene la resolución completa del taller de C++,
incluyendo todos los ejercicios solicitados y el informe en formato PDF.

 Estructura del repositorio

- **Ejercicios/**  
  Contiene todos los programas en C++ desarrollados para cada ejercicio
  del taller (Ejercicio01.cpp hasta Ejercicio41.cpp).

- **informe/**  
  Contiene el informe del taller en formato PDF, con introducción,
  desarrollo, capturas de pantalla y conclusiones.
